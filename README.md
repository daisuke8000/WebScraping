# webscp

## how to use

1. Create a virtual environment
```shell
$ python -m venv .venv
```

2. activate environment
```shell
$ source .venv/bin/activate
```

3. use package install in requirements.txt
```shell
$ pip install -r requirements.txt
```

4. run
```shell
$ python app/main.py
```
